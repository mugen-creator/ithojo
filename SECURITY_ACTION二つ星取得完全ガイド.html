🤔 SECURITY ACTIONって何？

一言で言うと...

「中小企業が情報セキュリティ対策に取り組んでいることを自己宣言する制度」です！

IPA（独立行政法人 情報処理推進機構）が運営する制度で、企業が情報セキュリティ対策に取り組むことを宣言し、ロゴマークを使用できるようになります。

📋 二つ星の取得要件

二つ星取得に必要な2つのステップ

ステップ1：「5分でできる！情報セキュリティ自社診断」を実施

IPAが提供する診断ツール（25項目）で自社のセキュリティ対策状況をチェックします。

  • 所要時間：約5〜10分
  • 診断結果をもとに、改善点を把握
  • 診断結果の提出は不要（自社で保管）

ステップ2：「情報セキュリティ基本方針」を策定し、外部公開

会社の情報セキュリティに対する基本的な方針を文書化し、ホームページなどで公開します。

  • IPAが提供するテンプレートを使用できる
  • 自社の状況に合わせてカスタマイズ
  • ホームページ、会社案内などで公開
  • 所要時間：1〜2時間
🎉 たったこれだけ！

この2つのステップを完了すれば、二つ星を宣言できます。
審査や認証機関の検査などは一切不要です。完全に自己宣言制なので、すぐに取得できます！

📝 二つ星の取得手順（完全版）

ステップバイステップで解説

1

SECURITY ACTION公式サイトにアクセス

Google検索で「SECURITY ACTION」と検索し、IPA公式サイトにアクセスします。
URL: https://www.ipa.go.jp/security/security-action/

2

「5分でできる！情報セキュリティ自社診断」を実施

IPAのサイトから診断ツールにアクセスし、25項目の質問に答えます。

診断項目の例：

  • パスワードを適切に管理していますか？
  • ウイルス対策ソフトを導入していますか？
  • OSやソフトウェアを最新状態に保っていますか？
  • 重要なデータのバックアップを取っていますか？
  • 従業員へのセキュリティ教育を実施していますか？

所要時間：5〜10分
診断結果は、自社のセキュリティ対策の現状把握に役立ちます。
診断結果の提出は不要です（自社で保管）。

3

診断結果を確認し、改善点を把握

診断結果をもとに、自社のセキュリティ対策の弱点を把握します。
この結果は、後で基本方針を策定する際の参考になります。

4

「情報セキュリティ基本方針」のテンプレートをダウンロード

IPAが提供する基本方針テンプレート（Word形式）をダウンロードします。
SECURITY ACTIONのサイトからダウンロードできます。

5

基本方針をカスタマイズ

テンプレートを自社の状況に合わせて編集します。

記載する主な内容：

  • 制定日
  • 会社名・代表者名
  • 目的（情報セキュリティの重要性を認識し、適切な対策を実施する、など）
  • 適用範囲（全従業員、協力会社を含む、など）
  • 基本方針
    • 情報資産の保護
    • 法令・規範の遵守
    • 継続的な改善
    • セキュリティ教育の実施
    • 事故発生時の対応
  • 責任者（代表取締役、情報セキュリティ責任者など）

所要時間：1〜2時間
テンプレートがあるので、初めてでも簡単に作成できます。

6

基本方針を外部公開

作成した基本方針を外部に公開します。

公開方法の例：

  • ホームページに掲載（最も一般的）
    「会社情報」「セキュリティポリシー」などのページに掲載
  • 会社案内に記載
  • 店舗やオフィスに掲示

ホームページがない場合でも、印刷して取引先に配布するなど、外部に見える形で公開すればOKです。

7

SECURITY ACTIONに申し込む

SECURITY ACTION公式サイトの「申込フォーム」から申し込みます。

必要な情報：

  • 会社名（商号）
  • 代表者名
  • 所在地
  • メールアドレス
  • 電話番号
  • 業種
  • 従業員数
  • 宣言する星の数（二つ星を選択）
  • 基本方針の公開URL（ホームページに掲載した場合）
8

自己宣言完了・ロゴ取得

申し込みが完了すると、SECURITY ACTIONのロゴマークがダウンロードできるようになります。

これで二つ星の取得完了です！

登録した情報は、IPAの「SECURITY ACTION宣言事業者一覧」に掲載されます。

9

ロゴマークを使用する

取得したロゴマークを様々な場所で使用できます：

  • 名刺（裏面に掲載する企業が多い）
  • ホームページ
  • 会社案内・パンフレット
  • 提案書・見積書
  • メールの署名
  • オフィスの掲示

📄 情報セキュリティ基本方針のサンプル

記載例

情報セキュリティ基本方針

制定日：2025年○月○日
株式会社○○○○
代表取締役 ○○ ○○

1. 目的

当社は、お客様からお預かりした情報や当社の事業活動において取り扱う情報を重要な資産と認識し、これらの情報を様々な脅威から守り、お客様や社会からの信頼に応えるため、情報セキュリティの確保に努めます。

2. 適用範囲

本方針は、当社の全役員及び全従業員に適用されます。また、当社と取引関係にある協力会社等にも、本方針の遵守を求めます。

3. 基本方針

(1) 情報資産の保護
当社が保有する情報資産を、不正アクセス、漏洩、改ざん、紛失、破壊等の脅威から保護します。

(2) 法令・規範の遵守
情報セキュリティに関する法令、国が定める指針、その他の規範を遵守します。

(3) 継続的な改善
情報セキュリティ対策を継続的に見直し、改善に努めます。

(4) セキュリティ教育の実施
全従業員に対し、情報セキュリティに関する教育・啓発活動を実施します。

(5) 事故発生時の対応
万一、情報セキュリティ事故が発生した場合は、迅速かつ適切に対応し、被害の最小化と再発防止に努めます。

4. 推進体制

代表取締役を情報セキュリティ責任者とし、情報セキュリティの維持・向上に取り組みます。

🔄 更新について

📅 年1回の更新が必要

SECURITY ACTIONは年1回の更新が必要です。

  • 更新時期：毎年、宣言した月
  • 更新方法：マイページから更新手続を行う（5分程度）
  • 更新費用：無料

更新を忘れると、宣言事業者一覧から削除され、ロゴマークも使用できなくなります。
更新時期が近づくと、登録したメールアドレスに通知が届くので安心です。

💡 更新時にやること

  • 情報セキュリティ基本方針の見直し（必要に応じて更新）
  • 自社診断の再実施（前回から改善されたか確認）
  • 従業員へのセキュリティ教育の実施状況確認

❓ よくある質問

Q1

本当に無料ですか？

はい、完全無料です。
申し込み、維持、更新、すべて無料です。
IPAが中小企業の情報セキュリティ対策を支援するために提供している制度なので、一切費用はかかりません。

Q2

審査はありますか？

審査はありません。
SECURITY ACTIONは自己宣言制です。
申し込みフォームを送信すれば、すぐにロゴマークが使用できるようになります。
認証機関の検査なども一切ありません。

Q3

ホームページがない場合はどうすればいいですか？

ホームページがなくても取得できます。
基本方針の公開方法は柔軟で、以下のような方法でもOKです：

  • 会社案内やパンフレットに掲載
  • オフィスや店舗に掲示
  • 取引先に配布
  • 無料ブログやSNSページに掲載

「外部に見える形で公開している」ことが重要です。

Q4

一つ星から二つ星にアップグレードできますか？

はい、いつでもアップグレードできます。
一つ星を取得した後、基本方針を策定・公開して、マイページから二つ星に変更できます。
追加費用は一切かかりません。

Q5

個人事業主でも取得できますか？

はい、取得できます。
SECURITY ACTIONは、法人だけでなく個人事業主も対象です。
屋号または個人名で申し込むことができます。

Q6

ロゴマークはどこに使えますか？

自社の宣伝・広報に自由に使用できます。

  • 名刺（裏面に掲載する企業が多い）
  • ホームページ
  • 会社案内・パンフレット
  • 提案書・見積書
  • メールの署名
  • 展示会のブース
  • オフィスの掲示

ただし、ロゴマークの改変は禁止されています。指定されたデザインをそのまま使用してください。

Q7

ISO27001（ISMS）との違いは？

SECURITY ACTIONは、ISMSの簡易版・入門版と言えます。

項目 SECURITY ACTION ISO27001
取得難易度 簡単 難しい
費用 無料 数十万円〜
期間 即日〜数時間 数ヶ月〜1年
審査 なし あり
信頼性 中 高

まずはSECURITY ACTIONで始めて、将来的にISO27001を目指すのもおすすめです。

Q8

更新を忘れたらどうなりますか？

宣言事業者一覧から削除され、ロゴマークも使用できなくなります。
ただし、再度申し込めば復活できます（何度でもOK）。
更新時期が近づくとメールで通知が来るので、忘れずに更新しましょう。

Q9

補助金申請で加点されますか？

補助金によっては加点対象になります。
特に以下の補助金では、SECURITY ACTIONの取得が有利に働くことがあります：

  • IT導入補助金
  • ものづくり補助金
  • 事業再構築補助金

補助金の公募要領で「情報セキュリティ対策の実施」が求められている場合、SECURITY ACTIONは有効なアピール材料になります。

Q10

取得後、何かやらなければいけないことはありますか？

特に義務付けられていることはありませんが、以下を継続的に実施することが推奨されます：

  • 情報セキュリティ基本方針に沿った運用
  • 従業員へのセキュリティ教育
  • 定期的な自社診断の実施
  • セキュリティ対策の継続的な改善

あくまで自己宣言制なので、厳格な義務はありませんが、宣言した以上は真摯に取り組むことが重要です。
